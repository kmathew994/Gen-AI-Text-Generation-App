# 🚀 Gen AI Text Generation App (Azure AI Foundry + Streamlit)

## 📌 Project Overview
This project is a **Generative AI-powered text generation application** that integrates a Large Language Model (LLM) with an interactive user interface. It demonstrates how modern AI systems can be deployed to create real-time, context-aware conversational experiences.

The application is designed to showcase **end-to-end AI integration**, including prompt engineering, parameter tuning, API-based model interaction, and frontend deployment.

---

## 🎯 Key Objectives
- Build a real-time AI-powered text generation system  
- Integrate Azure AI Foundry with a Python-based application  
- Design an intuitive chat interface for user interaction  
- Demonstrate practical use of prompt engineering and model tuning  
- Create a scalable foundation for AI-driven applications

## 🧠 Core Features
- 💬 **Interactive Chat Interface** – Real-time AI responses  
- 🎛️ **Dynamic Parameter Control** – Adjust temperature & top-p  
- 🧾 **Custom System Prompts** – Control tone and behavior of AI  
- 🧠 **Conversation Memory** – Maintains chat context  
- 📤 **Export Functionality** – Download chat history  
- ⚡ **Low-Latency API Integration** – Efficient response generation  

---

## 🏗️ System Architecture

User Input → Streamlit UI → Backend (Python) → Azure AI Foundry API → LLM Response → UI Display

---

## 🔍 How It Works
1. User inputs a prompt through the Streamlit interface  
2. The backend formats the request using prompt engineering techniques  
3. The request is sent to Azure AI Foundry’s Chat Completions API  
4. The LLM generates a context-aware response  
5. The response is displayed in real-time and stored in session history
